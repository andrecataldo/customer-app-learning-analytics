# Customer App – Learning Analytics (xAPI-inspired)

Pipeline analítico no Microsoft Fabric para logs de execução (execution_log_yyymmdd.csv),
com camadas Bronze → Silver → Gold → Semantic Model → Power BI, alinhado a princípios de xAPI/LRS.

## Estrutura
- notebooks/: notebooks exportados em .py
- sql/: views e checks
- docs/: arquitetura, contratos, TCC
- ml/: ML não supervisionado (exploratório)
- samples/: amostras pequenas e sanitizadas
